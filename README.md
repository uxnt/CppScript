## CppScript - 你好世界
```go
plug "stdio.h";
func main(){
     var x = 2021;
     print("Hello, World" + " - " + x);
}
```
打印输出：

```
Hello, World - 2021
```

## CppScript - 算法
```go
plug "stdio.h";
func main(){
     printf(1+1);
}
```

```
2
```

## CppScript - 判断
```go
plug "stdio.h"
func main(){
     var x = 6;
     if(x > 5){
          printf("Well done!");
     }
}
```
打印输出：
```
Well done!
```
